kdd.fitemset.fptree.fp_tree_algorithm module

kdd.fitemset.fptree.fp_tree_algorithm.retrieve_frequent_item_tuples(data_set, min_sup)[source]

Retrieve the frequent item tuples of a data set according to a minimum support.

Parameters:
  • data_set – list of integer lists
  • min_sup – the minimum amount for an item to be considered frequent
Returns:

a sorted list of item tuples that are frequent in the data set
Return type:

List[Tuple[int]]

Example:

>>> from kdd.fitemset.fptree.fp_tree_algorithm import retrieve_frequent_item_tuples

>>> data_set = [[1,2], [1,3], [1,4], [2], [2,3,4], (3,4)]
>>> min_sup = 2
>>> retrieve_frequent_item_tuples(data_set, min_sup)
[(1,), (2,), (3,), (3, 4), (4,)]
