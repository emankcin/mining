Commands after cloning:
```sh
$ cd mining
$ virtualenv venv
$ . venv/bin/activate
$ pip install pybuilder
$ pyb install_dependencies
$ pyb docs
```

Command to get out of virtual environment:
```sh
$ deactivate
```
