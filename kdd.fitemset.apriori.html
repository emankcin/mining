kdd.fitemset.apriori.apriori module

kdd.fitemset.apriori.apriori.apriori(data_set, min_sup, with_hash_tree=True)[source]

The apriori algorithm for retrieving frequent item tuples together with their counts.

Parameters:
  • data_set – A list of integer-lists (integers usually represent the keys of the items)
  • min_sup – minimum support, i.e. minimum number of items for them to be considered frequent
  • with_hash_tree (bool.) – True is default and means that the hash tree variant is used. False uses simple variant.
Returns:

dictionary of frequent item tuples with their counts
Return type:

Dict[Tuple[int] : int]

Example:

>>> from kdd.fitemset.apriori.apriori import apriori

>>> data_set = [[6], [7], [8], [3,6], [4,7], [5,8], [1,2,3], [7,8,9], [6,7,8,9]]
>>> min_sup = 3
>>> apriori(data_set, min_sup)
{(8,): 4, (6,): 3, (7,): 4}
>>> min_sup = 2
>>> apriori(data_set, min_sup)
{(8,): 4, (3,): 2, (7, 8, 9): 2, (9,): 2, (8, 9): 2, (6,): 3, (7,): 4, (7, 8): 2, (7, 9): 2}

kdd.fitemset.apriori.join module

kdd.fitemset.apriori.join.join(list1, list2)[source]

Join two integer lists. The join of two lists is the equal prefix of the two lists with both last elements sorted and appended.

Parameters:
  • list1 – list of integers
  • list2 – list of integers
Return type:

list of integers

Examples:

>>> from kdd.fitemset.apriori.join import join
>>> join([1,2,3], [1,2,4])
[1, 2, 3, 4]
>>> join([1,2,3], [1,3,4])
[]
>>> join([1,2,3], [1,1,4])
[]
>>> join([1,2,3], [1,2,3])
[]
>>> join([785, 43, 0], [785, 43, 1009])
[785, 43, 0, 1009]
>>> join([785, 43, 1009], [785, 43, 0])
[785, 43, 0, 1009]

Conditions for list1 and list2 to be joined:

>>> list1 = [1,2,3]
>>> list2 = [1,2,4]
>>> len(list1) == len(list2)
True
>>> len(list1) > 0
True
>>> list1[:-1] == list2[:-1]
True
>>> not list1[-1] == list2[-1]
True
